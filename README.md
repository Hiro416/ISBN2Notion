# ISBN2Notion 使い方

ISBN2Notionは、本のISBNバーコードを読み取って、書誌情報を確認し、Notionの蔵書Databaseへ登録するためのPWAです。

iPhoneのホーム画面に追加しておくと、買った本や手元の本をその場でNotionに入れられます。

## はじめて使うとき

1. アプリを開きます。
2. 合言葉を入力してログインします。
3. `Notionでログイン` を押します。
4. Notionの認可画面で、蔵書Databaseへのアクセスを許可します。
5. アプリに戻り、Notion接続が完了していることを確認します。

Notion接続はこの端末に保存されます。別の端末で使う場合は、その端末でもNotionログインを行ってください。

## 本を登録する

1. `バーコードを読み取る` を押します。
2. 本の裏表紙にある、978または979で始まるISBNバーコードをカメラに見せます。
3. 書誌情報が出たら、タイトル、著者、表紙、出版日などを確認します。
4. 必要に応じて `memo`、`Category`、`状態`、`Storage` を入力します。
5. `Notionに登録` を押します。

同じISBNの本が既にNotionにある場合は、新しく重複登録せず、登録済みとして扱います。

## ISBNを手入力する

カメラが使えない場合やバーコードを読みにくい場合は、`手入力ISBN` にISBNを入力して `探す` を押します。

ISBNはハイフンありでもなしでも入力できます。

## 書誌情報を手で入力する

書誌情報が見つからない本は、画面右上の `+` を押すと手入力で登録できます。

最低限、タイトルとISBNを入力してください。著者、表紙URL、出版日、出版社などは必要な範囲で入力できます。

## 入力欄の意味

| 項目 | 内容 |
| --- | --- |
| Title | 本のタイトル |
| Author | 著者、訳者など。複数ある場合はカンマ区切り |
| Cover | 表紙画像のURL |
| Published | 出版日または出版年 |
| ISBN | ISBN-10またはISBN-13 |
| Publisher | 出版社 |
| memo | 買った理由、読書メモ、登録時のひとこと |
| Category | 分類やタグ。複数ある場合はカンマ区切り |
| 状態 | `Unread`、`Reading`、`Finished` |
| Storage | 本の置き場所。`中野`、`仙台`、`電子` |

## うまくいかないとき

### カメラが起動しない

iPhoneやブラウザのカメラ権限を確認してください。PWAやSafariでカメラを使うには、HTTPSのURLで開く必要があります。

### ISBNではない数字を読む

日本の本には価格コードなど別のバーコードが並んでいることがあります。978または979で始まるISBNバーコードを狙ってください。

### Notionに登録できない

Notion接続が切れている可能性があります。`Notionでログイン` を押して、もう一度接続してください。

### 登録先Databaseが見つからない

Notion側のDatabaseに、アプリが必要とするプロパティが揃っているか確認してください。必要なプロパティは開発者に確認してください。

## ホーム画面に追加する

iPhoneのSafariでアプリを開き、共有メニューから `ホーム画面に追加` を選ぶと、通常のアプリのように起動できます。
